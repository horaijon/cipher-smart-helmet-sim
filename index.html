Helmet Detection

IR proximity sensor detects if helmet is properly worn before allowing ignition.

😴

Fatigue Monitoring

Eye-blink monitoring tracks rider alertness and triggers buzzer alerts on drowsiness.

🍺

Alcohol Detection

MQ-3 sensor analyzes breath to detect intoxication. Ignition locks if limits are exceeded.

💥

Accident Detection

MPU6050 accelerometer detects sudden impacts and triggers automatic SOS response.

📡

GPS + Emergency SMS

NEO-6M GPS + SIM800L GSM dispatch emergency SMS with live coordinates on crash.
